### `code_interpreter`（别名 `python`）
在有状态的 Jupyter 环境中执行代码。完整文档参见 `python` 工具。

### `browser`（已弃用）
这是早期的网页浏览工具。已被 `web` 替代。

### `my_files_browser`（已弃用）
旧版文件浏览器，用于暴露上传的文件以供浏览。已被自动文件内容暴露功能替代。

### `monologue_summary`
返回用户长篇独白的摘要。

用法：
```
monologue_summary: {
  content: string // 用户的完整消息
}
```

返回如下摘要：
```
{
  summary: string
}
```

### `search_web_open`
将 `web.search` 和 `web.open_url` 合并为单次调用。

用法：
```
search_web_open: {
  query: string
}
```

返回：
```
{
  results: string // 顶部搜索结果的提取内容
}
```
